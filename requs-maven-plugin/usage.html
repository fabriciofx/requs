Usage with Maven 3+

Add the following to your pom.xml:

[...]
<build>
  [...]
  <plugins>
    [...]
    <plugin>
      <artifactId>org.requs</artifactId>
      <groupId>requs-maven-plugin</groupId>
      <version>1.2</version>
    </plugin>
  </plugins>
</build>

Running mvn requs:compile will create compile reports in project build directory, inside requs sub-directory.

Add the following to your pom.xml (Maven 3.0+):

[...]
<build>
  [...]
  <plugins>
    [...]
    <plugin>
      <artifactId>maven-site-plugin</artifactId>
      <configuration>
        <reportPlugins combine.children="append">
          <plugin>
            <groupId>org.requs</groupId>
            <artifactId>requs-maven-plugin</artifactId>
            <version>1.2</version>
          </plugin>
        </reportPlugins>
      </configuration>
    </plugin>
  </plugins>
</build>

Running mvn site will create a summary report in target/site/requs.
