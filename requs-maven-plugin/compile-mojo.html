requs:compile

Full name:

org.requs:requs-maven-plugin:1.11:compile

Description:

Compile requs sources into XML.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 1.1.
  • Binds by default to the lifecycle phase: compile.

Required Parameters

Name Type Since Description
input File 1.1 The source directory.
Default value is: ${basedir}/src/main/requs.
output File 1.1 Output directory.
Default value is: ${project.build.directory}/requs.
